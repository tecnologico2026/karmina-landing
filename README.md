# Karmina la Capibara — Tomo 1

Landing page for the children's book *Karmina la Capibara — Tomo 1: Historias de la selva amazónica* by Jaime Río.

Static HTML/CSS site hosted on GitHub Pages.

## Live URL

https://tecnologico2026.github.io/karmina-landing/

## Stack

- HTML5 semantic
- CSS3 (no frameworks)
- Google Fonts: Cormorant Garamond + Lora
- Mobile-first responsive

## Structure

```
karmina-landing/

```
